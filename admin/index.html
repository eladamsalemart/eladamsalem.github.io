כדי לערוך ולפרסם שינויים, התחבר פעם אחת עם מפתח גישה אישי של GitHub (Token). הוא נשמר רק בדפדפן שלך.

איך משיגים מפתח (פעם אחת, ~דקה):
  1. פתח: יצירת Token ב‑GitHub ↗
  2. תן שם כלשהו (למשל site editor), ותחת Repository access בחר Only select repositories → eladamsalem.github.io.
  3. תחת Permissions → Repository permissions → Contents בחר Read and write.
  4. לחץ Generate token, העתק אותו, והדבק כאן למטה.
